# grokbot

A programmable humanoid robot project using a 17 DOF (Degrees of Freedom) biomimetic robot frame.

## Hardware

### Robot Frame

**17 DOF Biomimetic Humanoid Robot Frame**

- **Price**: $137.04
- **Servos**: Dual-axis RDS3115 servos
- **DOF**: 17 degrees of freedom
- **Purchase Link**: [AliExpress](https://a.aliexpress.com/_mMKESe1)

#### Features

- Biomimetic design for natural humanoid movement
- High degree of freedom (17 DOF) for complex motions
- Dual-axis servo support (RDS3115)
- Suitable for various applications:
  - Fighting competitions
  - Dance routines
  - Bipedal locomotion research
  - Programmable robot projects

#### Technical Specifications

- **Degrees of Freedom**: 17 DOF
- **Servo Model**: RDS3115 (dual-axis capable)
- **Frame Type**: Biomimetic humanoid
- **Applications**: Competition fighting, dancing, bipedal walking

## Getting Started

This project is designed to work with the 17 DOF humanoid robot frame listed above. The robot can be programmed for various activities including competitive fighting, choreographed dance routines, and bipedal locomotion experiments.
